Message du développeur : 

L'utilisation de ce logiciel est assez compliquée donc voici quelques explications qui peuvent être utiles : 

	- cette version est minimaliste pour ne pas vous obliger à installer d'autres logiciels complémentaires. Il ne corrige donc pas les fautes mais simule juste une présence humaine.

	- au démarrage (après avoir fermé les menus d'aides) du logiciel, vous devez ouvrir votre navigateur (Google Chrome, Firefox, Tor, ...) et vous connectez sur le site. Sélectionnez ensuite un des niveaux comme si vous vouliez vous entrainer.

	- ce logiciel ne sait pas la première fois où se trouvent les boutons, les textes, ... c'est à vous de le lui dire. Pour ce faire, retournez sur la fenêtre du logiciel et appuyez sur la touche e, au bout de quelques secondes le titre passe en vert. Attention ! Si vous ne bougez plus votre souris pendant 1 seconde environ (cela dépend des ordinateurs) vous entendrez la douce voix d'un ami ce qui signifie que la position sera enregistrée ! C'est sur ce principe que vous devez configurer la position des boutons ou du texte sur le site. Une vidéo d'aide : aide.mp4 est disponible dans ce dossier. Quand vous avez fini de configurer les positions, rappuyez sur e (en étant bien sur la fenêtre du logiciel). À présent, vous avez fait le plus dur, si l'option tout en bas est activée (et en vert), alors au prochain démarrage du logiciel, vos positions seront réutilisées. Vous pouvez même les rajouter au fur et à mesure (en rappuyant sur e).

	- Pour commencer la simulation, il vous suffit d'appuyer sur s (en étant sur la fenêtre). Vous avez alors quelques secondes pour vous mettre sur votre navigateur et admirer cette simulation. Ce processus fait de longue pause pour vous laisser le temps de l'arrêter. Pour ce faire, quand la souris ne bouge plus, retournez sur la fenêtre et rappuyez sur s.

Ce message apparait que si l'option menu d'aide est activée (touche m).
